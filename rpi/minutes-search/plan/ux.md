# UX Design: minutes-search

**Feature**: 過去議事録検索
**Date**: 2026-03-17

---

## User Flow

### Flow 1: 議事録検索

```
ユーザー: /minutes search keyword:予算
   ↓
Bot: (ephemeral) 検索結果Embed表示
   - 結果1: 2026-03-10 会議 — "...来期予算について議論し..."
   - 結果2: 2026-02-28 会議 — "...予算配分の見直しを..."
   - フッター: "2件中 2件を表示"
```

### Flow 2: 検索結果なし

```
ユーザー: /minutes search keyword:存在しないキーワード
   ↓
Bot: (ephemeral) "「存在しないキーワード」に一致する議事録は見つかりませんでした。"
```

### Flow 3: アーカイブが空

```
ユーザー: /minutes search keyword:テスト
   ↓
Bot: (ephemeral) "まだアーカイブされた議事録がありません。議事録が生成されると自動的にアーカイブされます。"
```

---

## Discord Command Interface

### `/minutes search`

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| `keyword` | string | Yes | 検索キーワード |

## Search Result Embed Design

```
┌─────────────────────────────────────────────┐
│  🔍 議事録検索結果 — 「予算」               │
│  color: #5865F2                              │
├─────────────────────────────────────────────┤
│                                             │
│  📅 2026-03-10 — 参加者: Alice, Bob         │
│  ───────────────────────────────            │
│  ...来期**予算**について議論し、各部門の     │
│  配分を決定した...                          │
│                                             │
│  📅 2026-02-28 — 参加者: Alice, Charlie     │
│  ───────────────────────────────            │
│  ...**予算**配分の見直しを行い、開発部門    │
│  への増額を承認した...                      │
│                                             │
├─────────────────────────────────────────────┤
│  2件中 2件を表示                            │
└─────────────────────────────────────────────┘
```

### Embed構成

- **Title**: `🔍 議事録検索結果 — 「{keyword}」`
- **Color**: `#5865F2`（既存のminutes embedと統一）
- **Fields**: 各検索結果を1フィールドとして表示
  - **Name**: `📅 {date_str} — 参加者: {speakers}`
  - **Value**: FTS5 snippet（マッチ箇所前後テキスト）
- **Footer**: `{表示件数}件中 {全件数}件を表示`

### 制約
- 最大5件表示（Discord Embedフィールド数制限を考慮）
- 各スニペットは最大200文字
- Embed全体は4000文字以内
- 応答は **ephemeral**（議事録内容のプライバシー保護）

---

## Error States

| 状態 | 表示 |
|------|------|
| キーワード空文字 | Discord側バリデーションで防止（required parameter） |
| 検索結果なし | 「「{keyword}」に一致する議事録は見つかりませんでした。」 |
| アーカイブ未初期化 | 「まだアーカイブされた議事録がありません。」 |
| 検索エラー | 「検索中にエラーが発生しました。しばらくしてからお試しください。」（ephemeral） |

---

## Accessibility

- ephemeral応答でプライバシー保護（他メンバーに検索内容が見えない）
- スニペット内のマッチ箇所はプレーンテキスト（Discordはboldをサポート）
- 日本語UI統一（既存コマンドと同じ言語）
