# UX Plan: 詳細議事録フォーマット + トランスクリプト添付

## Discord投稿の構成

### Embed（変更なし）
```
┌─────────────────────────────────┐
│ 会議議事録 — 2026-03-16 14:01   │
├─────────────────────────────────┤
│ 参加者: Alice, Bob              │
│                                 │
│ 要約:                           │
│ （3〜5行の概要）                 │
│                                 │
│ 決定事項:                       │
│ - 項目1                         │
│ - 項目2                         │
│                                 │
│ 話者統計:                       │
│ alice ████████░░ 8:32           │
├─────────────────────────────────┤
│ 詳細議事録は添付ファイルを参照   │
└─────────────────────────────────┘
```

### 添付ファイル1: 議事録 (`minutes_YYYY-MM-DD_HHMM.md`)

```markdown
# 会議議事録
- 日時: 2026-03-16 14:01
- 参加者: Alice, Bob

## まとめ

（3〜5行の概要テキスト）

**トピック1のタイトル**
トピック1の要約（1〜2文）

**トピック2のタイトル**
トピック2の要約（1〜2文）

## 詳細

* **トピックの要点**: 詳細な説明文。([00:04:45])
* **トピックの要点**: 詳細な説明文。([00:07:25])
...（30+項目）

## 推奨される次のステップ

- [ ] 担当者は〇〇を行います。
- [ ] 担当者は〇〇を完了します。
```

### 添付ファイル2: トランスクリプト (`transcript_YYYY-MM-DD_HHMM.md`)

```markdown
# 文字起こし
- 日時: 2026-03-16 14:01
- 参加者: Alice, Bob

### 00:00:00

**Alice:** テキスト
**Bob:** テキスト

### 00:03:03

**Alice:** テキスト
...
```

## Embed正規表現の対応

- `_SUMMARY_PATTERN`: `## 要約` → `## まとめ` に変更
- `_DECISIONS_PATTERN`: `## 決定事項` → `## 推奨される次のステップ` に変更
- Embedフィールド名も対応: 「要約」→「まとめ」、「決定事項」→「次のステップ」

## ForumChannel / TextChannel 別挙動

| チャネル | Embed | ファイル |
|----------|-------|---------|
| ForumChannel | create_thread(embed) | thread.send(files=[議事録, トランスクリプト]) |
| TextChannel | send(embed, files=[議事録, トランスクリプト]) | — |
